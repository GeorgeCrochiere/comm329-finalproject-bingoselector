    About

    close_fullscreen

    This site is a random item selector, typically used for games of Bingo, that allows for the removal of items with later options to reset the game back to the original, default state. It does so by storing the inputted list into the browser's session storage.

    This site was developed as the final project for George D. Crochiere for the COMM329 course at Clarkson University, with the intend of developing a tool to help the Clarkson Honors Program.
